HTML Quotations

Tag Description Output Example
<abbr> Defines an abbreviation or acronym <abbr title="Hypertext Markup Language">HTML</abbr>
Output: HTML
<address> Defines contact information for the author/owner of a document <address>123 Main St, Anytown, USA</address>
Output: 123 Main St, Anytown, USA
<bdo> Defines the text direction <bdo dir="rtl">This text will be displayed from right to left</bdo>
Output: This text will be displayed from right to left
<blockquote> Defines a section that is quoted from another source <blockquote>This is a quote from another source</blockquote>
Output: This is a quote from another source
<cite> Defines the title of a work <cite>The Great Gatsby</cite>
Output: The Great Gatsby
<q> Defines a short inline quotation <q>This is a short quote</q>
Output: "This is a short quote"
