Tag Description Example with Nested Tags
<ul> <li> Defines an unordered list <ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
Output:
  • Item 1
  • Item 2
  • Item 3
<ol> <li> Defines an ordered list <ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>
Output:
  1. First
  2. Second
  3. Third
<dl> <dt> <dd> Defines a description list <dl>
  <dt>Term 1</dt>
  <dd>Definition 1</dd>
  <dt>Term 2</dt>
  <dd>Definition 2</dd>
</dl>
Output:
Term 1
Definition 1
Term 2
Definition 2
